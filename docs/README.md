## ✍️ 文档规范

### 更新原则
1. **及时更新**: 代码变更后同步更新文档
2. **准确完整**: 确保文档内容准确、完整
3. **简洁明了**: 使用清晰的语言和示例
4. **版本标记**: 文档底部标记最后更新日期

### Markdown规范
- 使用清晰的标题层次
- 代码块注明语言
- 使用表格整理数据
- 添加emoji增强可读性
- 使用相对路径链接

### 文档模板
每个主要文档应包含：
```markdown
# 标题

简介说明

## 主要内容
...

---

**最后更新**: YYYY-MM-DD
```

## 🔍 搜索技巧

在项目根目录使用grep搜索文档：
```bash
# 搜索特定内容
grep -r "关键词" docs/

# 搜索文件名
find docs/ -name "*关键词*.md"
```
